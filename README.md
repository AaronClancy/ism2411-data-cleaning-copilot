# CSV Data Cleaner
Asks the user to input the number of hours they spend studying per week. It checks for input 
validity with a "try" statement and converts the input to a floating point value. It then 
multiplies the user inputted nunmber by 7 to find the average weekly study hours. It outputs 
this number in an output message.

# How to Run
```
python data-cleaning.py
```
or
```
python3 data-cleaning.py
```
